# Diagrame PULSE

Acest folder indexeaza diagramele proiectului PULSE.

Diagramele UML, arhitectura componentelor si workflow-urile sunt mentinute in format Mermaid in:

- `../diagrame_uml_pulse.md`
- `../diagrame_uml_arhitectura_workflowuri.md`

Documentele acopera:

- diagrame de cazuri de utilizare pentru utilizator/medic si administrator;
- diagrame de clase pentru content, events, courses, users, ads, subscriptions si payments;
- diagrama de pachete;
- diagrame de secventa pentru register, autentificare, publicare continut, inscriere la eveniment si submisii;
- diagrame de stari pentru `ContentItem`, `Ad`, `UserSubscription`, `UserEventRegistration` si `ContentSubmission`;
- diagrama de arhitectura a componentelor si deployment.

Diagramele pot fi vizualizate direct in GitHub, deoarece blocurile Mermaid sunt randate automat in fisierele Markdown de mai sus.
